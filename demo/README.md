# Cinematic Decision Demo

这是一个独立于现有业务代码的 UI 原型，用来评审“电影化二选一抉择界面”的质量。

## 打开方式

直接双击 `index.html` 即可。

如果浏览器拦截本地视频自动播放，可以在项目根目录启动一个临时静态服务后访问 `demo/index.html`。

## 已包含效果

- 参考视频作为中间决策带背景
- 左右双选项布局
- 中央准星与发光核心
- 红色扫描光束与拖影
- 鼠标移动时准星追踪
- Hover 聚焦
- 键盘切换与确认态
- 移动端纵向重排

## 交互说明

- `A` / `←`：选择左侧
- `D` / `→`：选择右侧
- `Enter` / `Space`：确认当前选项
- `Esc`：重置确认状态

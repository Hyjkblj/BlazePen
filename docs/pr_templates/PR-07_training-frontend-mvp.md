# PR-07 训练主线前端 MVP 接入

## 目标

把训练主线从后端能力升级为可用前端产品主流程。

## 依赖

- `PR-02`
- `PR-05`

## 本次前端必须完成

- 新增训练入口
- 新增训练模式选择与训练场景页
- 接入训练初始化、下一场景、回合提交接口
- 将训练流程与故事流程在业务层解耦

## 本次后端必须完成

- 校正训练接口文档与真实返回
- 补齐前端 MVP 需要的字段
- 如有必要，新增训练会话摘要接口

## 本次不做

- 不做训练报告和诊断页
- 不做复杂的数据分析看板
- 不做故事流程重构

## 契约要求

- 训练只使用 `sessionId`
- 回合响应必须显式返回 `roundNo / isCompleted / nextScenario / evaluation / runtimeState`

## 测试要求

### 前端

- 训练初始化测试
- 下一场景获取测试
- 单回合提交测试
- 训练刷新恢复测试

### 后端

- 训练初始化接口测试
- 下一场景接口测试
- 回合提交接口测试

## Reviewer Focus

### 前端

- 训练业务逻辑是否与故事逻辑隔离
- 是否滥用已有故事 hook

### 后端

- DTO 是否稳定可产品化
- 是否存在前端接入所需字段缺失

## 验收标准

- 前端能完整跑通训练 MVP 主流程
- 故事与训练状态互不污染
- 刷新后训练会话可恢复

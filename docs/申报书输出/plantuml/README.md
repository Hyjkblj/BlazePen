# BlazePen PlantUML 配图源文件

本目录包含申报书配图的 PlantUML 源文件：

1. `01_项目总览图.puml`
2. `02_场景库结构图.puml`
3. `03_能力指标图.puml`
4. `04_联合评估机制图.puml`
5. `05_知识追踪图.puml`
6. `06_技术架构图.puml`
7. `07_系统界面结构示意图.puml`

对应导出的 SVG 文件位于 `svg/` 目录：

1. `svg/01_项目总览图.svg`
2. `svg/02_场景库结构图.svg`
3. `svg/03_能力指标图.svg`
4. `svg/04_联合评估机制图.svg`
5. `svg/05_知识追踪图.svg`
6. `svg/06_技术架构图.svg`
7. `svg/07_系统界面结构示意图.svg`

对应导出的 PNG 文件位于 `png/` 目录：

1. `png/01_项目总览图.png`
2. `png/02_场景库结构图.png`
3. `png/03_能力指标图.png`
4. `png/04_联合评估机制图.png`
5. `png/05_知识追踪图.png`
6. `png/06_技术架构图.png`
7. `png/07_系统界面结构示意图.png`

说明：

- 这些文件可继续编辑，用于后续导出 PNG 或 SVG。
- 若需真实系统截图版“系统界面图”，需要基于现有前端页面另行整理。
